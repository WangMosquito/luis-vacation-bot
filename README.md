# luis-vacation-bot
使用 LUIS + Bot Framework 打造請假對談機器人，其中 [LUIS](http://luis.ai/) 負責解析語言的部份，而 [Bot Framework](https://dev.botframework.com/) 則是完成對談機器人的製作。
